Install

The first step is to get oneself a Windows 10 based operating system, preferably one that is suited to gaming or high-end visualization. If you have a MacBook Pro, you could either use Bootcamp or Parallels to setup a Windows 10 operating system. What is noteworthy is that Microsoft offers Windows 10 evaluation version -- valid for three months -- if all you want is to try it out! Once you have a Windows 10 operating system on your computer, enable Hyper V. To do this, open control panel and then navigate to : Programs > Programs and Features > Turn Windows Features on or off > Hyper-V.
